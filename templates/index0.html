about.

ColorBorg is an automation platform for Old School RuneScape. Instead of using injection to interact with the game, ColorBorg employs a mix of color detection, image recognition, optical character recognition(OCR), and a local API to navigate the game successfully. The platform is developed in Python with a focus on modularity and ban evasion. Throughout its development, I designed the bot to mimic human behavior, enhancing its natural appearance and reducing the risk of detection.

The system's modularity plays a significant role in its flexibility; most functions are written to be universal, allowing them to be reused for other skills. This plug-and-play approach helps ensure that the bot is easily adaptable and expandable for additional in-game tasks.

features.

The task and skills below are the currently supported skills and task Colorborg can do. More to come in the future!

  • Fisher: Supports shrimp and fly fishing
  • Slayer: Slays Chickens to collect their feathers
  • Walker: Able to traverse the game world autonomously

Below are the features that truly allow colorborg to be best in class.

  • Runelite Local API: Utilizes the RuneLite local API to optimize computational performance and achieve tasks otherwise impossible with a standard color bot.
  • GUI: A clean, straightforward interface providing all necessary controls without unnecessary clutter.
  • Logging: Comprehensive logging capabilities to facilitate efficient debugging and ongoing support.
  • Break System: Allows the bot to take breaks automatically, enhancing its human-like behavior and reducing the risk of detection.
  • Ban Evasion: Designed with ban evasion as a core focus, incorporating the latest techniques to minimize the risk of detection by anti-botting systems.

about the site.

Design and Frontend
I designed this website to be colorful and vibrant, paying homage to the name of the bot, the program it showcases. The frontend is built using HTML and CSS, ensuring a responsive and visually appealing experience across all devices. My goal was to create a user-friendly interface that effectively highlights my projects and skills while reflecting the essence of Colorborg. Backend and Security
Security is a top priority in my development process. The backend of this website is powered by Flask, with robust security measures implemented throughout. Passwords are securely hashed and salted, and sensitive areas of the site are protected by role-based permissions. I coded the admin panel from scratch, ensuring it is accessible only to authorized users.
